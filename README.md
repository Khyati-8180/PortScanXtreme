# 🔐 Security Scanner Tool

A Python-based multi-threaded security scanner that performs port scanning, service detection, and basic vulnerability analysis.

## 🚀 Features
- Multi-threaded port scanning
- Banner grabbing (service detection)
- Security header analysis
- Color-coded CLI output

## 🛠️ Technologies Used
- Python
- Socket Programming
- Threading
- Requests
- Colorama

## 📌 How to Run
1. Install dependencies:
   pip install requests colorama

2. Run the script:
   python scanner.py

## 🎯 Output
- Shows open/closed ports
- Displays running services
- Highlights missing security headers
